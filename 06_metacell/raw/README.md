# Raw Data — Metacell

The processing script (`processing/mcrigor_intro_vignette.qmd`) reads data
bundled with the `mcRigor` R package via `system.file()` — no files need to be
manually placed here - but the package needs to be installed first.

Outputs (`mcrigor_intro_vignette.h5ad` and `mcrigor_intro_vignette.rds`) are written to `../data/` by `processing/mcrigor_intro_vignette.qmd`.
