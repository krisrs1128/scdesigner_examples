# Raw Data — Spatial Domains

The processing script reads 26 SpaceRanger output tarballs and writes filtered objects to `data/`.

Paper: [https://www.nature.com/articles/s41593-024-01765-6](https://www.nature.com/articles/s41593-024-01765-6)
Geo: [https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE208747](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE208747)

| File | Role | Source |
|------|------|--------|
| `GSM636879[3-6]_A[1-4].tar.gz` | **Input** — SpaceRanger outputs for samples A1–A4 | [GEO GSM6368793–GSM6368796](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSM6368793) |
| `GSM636879[7-8]_C[1-2].tar.gz` | **Input** — SpaceRanger outputs for samples C1–C2 | [GEO GSM6368797–GSM6368798](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSM6368797) |
| `GSM636879[9]_CG1.tar.gz`, `GSM636880[0-1]_CG[2-3].tar.gz` | **Input** — SpaceRanger outputs for samples CG1–CG3 | [GEO GSM6368799–GSM6368801](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSM6368799) |
| `GSM636880[4-9]_M[1-6].tar.gz` | **Input** — SpaceRanger outputs for samples M1–M6 | [GEO GSM6368804–GSM6368809](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSM6368804) |
| `GSM636881[0-2]_N[1-3].tar.gz` | **Input** — SpaceRanger outputs for samples N1–N3 | [GEO GSM6368810–GSM6368812](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSM6368810) |
| `GSM636881[3-5]_NG[1-3].tar.gz` | **Input** — SpaceRanger outputs for samples NG1–NG3 | [GEO GSM6368813–GSM6368815](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSM6368813) |
| `GSM636881[8-9]_SG[1-2].tar.gz`, `GSM636882[0-2]_SG[3-5].tar.gz` | **Input** — SpaceRanger outputs for samples SG1–SG5 | [GEO GSM6368818–GSM6368822](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSM6368818) |
| `*.jpg.gz`, `*.json.gz` | **Input** — high-resolution tissue images and spatial alignment metadata (one pair per sample) | same GEO accessions above |
| `data/spes_filtered.rds` | **Output** — list of filtered `SpatialExperiment` objects, generated by `processing/process.qmd` | — |
| `data/h5ad/{sample}.h5ad` | **Output** — per-sample AnnData files, generated by `processing/process.qmd` | — |

Place all `.tar.gz`, `.jpg.gz`, and `.json.gz` files here before running the processing script.
