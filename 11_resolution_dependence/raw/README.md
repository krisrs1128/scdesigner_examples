# Raw Data — Resolution Dependence

The processing script reads one input file and writes two output files stored in `data/`.

| File | Role | Source |
|------|------|--------|
| `merfish.csv` | **Input** — MERFISH dataset with cell metadata and per-gene counts (cells × features) | [Cable et al. 2022](https://www.nature.com/articles/s41592-022-01575-3#data-availability) and [Moffit et al.](https://datadryad.org/dataset/doi:10.5061/dryad.8t8s248) |
| `data/merfish.rds` | **Output** — `SpatialExperiment` object, generated by `processing/process.qmd` | — |
| `data/merfish.h5ad` | **Output** — AnnData version, generated by `processing/process.qmd` | — |

Place `merfish.csv` here before running the processing script.
