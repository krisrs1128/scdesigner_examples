# Raw Data — Differential Expression Benchmark

The processing script reads one input file from this directory and writes outputs to `../data/`.

| File | Role | Source |
|------|------|--------|
| `SRP162188.rds` | **Input** — raw single-cell RNA-seq `SingleCellExperiment` object | `smb://research.drive.wisc.edu/ksankaran/scdesigner/data/04_differential_expression_benchmark/SRP162188.rds` |

Outputs (`SRP162188.h5ad` and `SRP162188.rds`) are written to `../data/` by `processing/process.qmd`.

Place `SRP162188.rds` here before running the processing script.
